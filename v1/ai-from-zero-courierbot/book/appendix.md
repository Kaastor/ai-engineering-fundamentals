# Appendix — Topics intentionally de-emphasized in live hours

These topics are valuable, but the course delays them until you have a working agent skeleton
and have seen real failure modes.

## Search extras

- DFS and UCS details
- admissibility proofs and complexity bounds

## Probability extras

- Markov property formalism
- POMDP notation

## Learning extras

- Bellman equations
- deep net internals and function approximation theory

## Engineering extras

- circuit breakers
- concurrency rules
- full security architecture

## “Adult invariants” checklist (reading)

- Effect typing (what side effects exist?)
- Operability (logs, metrics, alarms)
- Reproducibility discipline (seeds, artifacts, environments)
- Evaluation hygiene (fixed suites, regression gates, adversarial tests)
